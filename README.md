# Namma Yatri: Rider Funnel & Cancellation Analysis

An analysis of Namma Yatri's rider journey, focused on conversion funnel drop-offs and booking cancellations.

## Key Findings
- Only 24.4% of searches become completed rides
- Biggest drop: Search → Quote (47.2% lost)
- 72% of driver cancellations happen when pickup distance exceeds 1km
- NOT_REACHABLE is the #1 cancellation reason across all time slots
